# AI--HELATHCARE-CHATBOT
MedAssist Pro is an AI-powered healthcare assistant designed to help users understand their medical reports, receive personalized health insights, and locate nearby healthcare facilities. 
MedAssist Pro is an AI-powered healthcare assistant designed to help users understand their medical reports, receive personalized health insights, and locate nearby healthcare facilities. The platform combines Retrieval-Augmented Generation (RAG) with the Google Gemini Large Language Model to provide accurate, context-aware responses based on uploaded medical documents.

Users can upload medical reports in PDF format, which are processed through a RAG pipeline involving text extraction, document chunking, embedding generation, and vector database storage. When a user asks a question, the system retrieves the most relevant information from the uploaded documents and uses Gemini AI to generate reliable answers supported by source citations.

The platform also includes a hospital finder feature that helps users discover nearby hospitals and clinics through an interactive map interface. A centralized dashboard allows users to track reports analyzed, conversations, health recommendations, and healthcare activities.
